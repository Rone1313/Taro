# Море мудрости — сайт (GitHub Pages)

Папка готова к публикации на GitHub Pages. Внутри:

- `index.html` — интерактивная карта (картинки встроены).
- `more-mudrosti-original.pdf` — оригинал (для кнопки «Открыть оригинал PDF»).
- `.nojekyll` — отключает обработку Jekyll (нужно для статики).

## Публикация через Git

В терминале из этой папки:

```bash
git init
git add .
git commit -m "Mind map: Море мудрости"
git branch -M main
git remote add origin https://github.com/ВАШ-ЛОГИН/ВАШ-РЕПО.git
git push -u origin main
```

Затем в репозитории на GitHub: **Settings → Pages → Source: Deploy from a branch → Branch: `main` / `/ (root)` → Save.**
Через минуту сайт будет доступен по адресу `https://ВАШ-ЛОГИН.github.io/ВАШ-РЕПО/`.

## Пароль

Вход в карту защищён паролем (мягкий замок на стороне браузера). По умолчанию: `nagarjuna2026`.
Чтобы сменить — в `gen_map2.py` поменяйте строку `PASSWORD = "..."`, пересоберите и замените `index.html`.
Важно: это деферрент от случайных, а не строгая защита (на публичном Pages исходник доступен). Для строгого доступа закрытой группы — Cloudflare Access (по списку email).

## Общие заметки/рисунки (бэкенд)

GitHub Pages — статический хостинг, своего бэкенда нет. Сейчас заметки и рисунки хранятся локально в браузере каждого. Чтобы они были общими для группы, подключается внешний сервис (рекомендуется Supabase, бесплатно) — это отдельная доработка кода карты.
